# Инструкция по установке зависимостей для проекта

## Текущие версии
- Node.js: v20.10.0 ✅ (подходит)
- npm: 10.2.3 ✅ (подходит, обновление не требуется)
- Java: 17.0.16 ❌ (нужна версия 21+)

## Что нужно установить

### 1. Java Development Kit (JDK) 21 или выше

**Проблема:** Текущая версия Java 17 не поддерживает class file version 65.0, который требуется для shadow-cljs 2.26.2.

**Решение:** Установите JDK 21 (LTS версия)

**Варианты установки:**

#### Вариант A: Oracle JDK 21
1. Скачайте с официального сайта: https://www.oracle.com/java/technologies/downloads/#java21
2. Установите и настройте переменную окружения `JAVA_HOME`

#### Вариант B: OpenJDK 21 (рекомендуется)
1. Скачайте Eclipse Temurin 21: https://adoptium.net/temurin/releases/?version=21
2. Или используйте Chocolatey (если установлен):
   ```powershell
   choco install temurin21
   ```

#### Вариант C: Использование SDKMAN (если доступен)
```bash
sdk install java 21.0.1-tem
```

**После установки Java 21:**
1. Проверьте версию:
   ```powershell
   java -version
   ```
   Должно показать версию 21.x.x

2. Убедитесь, что `JAVA_HOME` указывает на Java 21:
   ```powershell
   echo $env:JAVA_HOME
   ```

### 2. Node.js (опционально, если хотите обновить)

**Текущая версия v20.10.0 работает**, но если хотите обновить:

**Вариант A: Обновить до Node.js 20.17.0+ (LTS)**
1. Скачайте с https://nodejs.org/
2. Или используйте nvm-windows:
   ```powershell
   nvm install 20.17.0
   nvm use 20.17.0
   ```

**Вариант B: Обновить до Node.js 22.9.0+ (Current)**
1. Скачайте с https://nodejs.org/
2. Или используйте nvm-windows:
   ```powershell
   nvm install 22.9.0
   nvm use 22.9.0
   ```

**Важно:** Обновление Node.js НЕ обязательно! Текущая версия v20.10.0 с npm 10.2.3 работает с shadow-cljs 2.26.2.

## После установки Java 21

1. Перезапустите терминал/PowerShell
2. Проверьте версии:
   ```powershell
   java -version
   node --version
   npm --version
   ```

3. Установите зависимости проекта:
   ```powershell
   cd lab4/my-app
   npm install
   ```

4. Запустите проект:
   ```powershell
   npx shadow-cljs watch app
   ```

## Примечания

- **НЕ нужно** обновлять npm до 11.7.0 - текущая версия 10.2.3 работает нормально
- **НЕ нужно** обновлять Node.js, если Java 21 решит проблему
- Основная проблема - это версия Java, а не Node.js/npm
